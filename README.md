# Snake Game (Terminal)

A tiny terminal snake game written in Rust.
# hugo & abdallah

## Run

```bash
cargo run -p snake_game
```

## Test

```bash
cargo test
```

## Notes

- Controls: `W/A/S/D` move, `M` mode, `T` speed, `P` pause, `R` restart, `Q` quit.
- The game uses deterministic pseudo-random spawning for food and rocks.
